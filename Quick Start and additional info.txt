Quick Start
python

from trpl_analyzer import TRPLDataGenerator, TRPLFitter, TRPLPlotter

# Generate simulated data
t = TRPLDataGenerator.create_time_axis(num_points=500)
decay_true, decay_noisy = TRPLDataGenerator.generate_decay(
    t, params={'A1': 1000, 'tau1': 2, 'A2': 500, 'tau2': 20}, 
    model='biexponential'
)

# Fit the data
fitter = TRPLFitter()
fit_result = fitter.fit_scipy(t, decay_noisy, model='biexponential')

# Plot results
plotter = TRPLPlotter(style='publication')
plotter.plot_decay(t, decay_noisy, fit_result=fit_result)

Key Features Implemented:

    Simulated Data Generation: Create decays with realistic noise

    Multi-exponential Fitting: Mono-, bi-, and tri-exponential models

    Two Fitting Backends: SciPy (simple) and LMFit (advanced)

    Publication-quality Plots: Customizable matplotlib styles

    Data I/O: Load/save various formats

    Parameter Analysis: Calculate lifetimes, amplitudes, averages

    Residual Analysis: Diagnostic plots for fit quality

    Component Visualization: Show individual exponential contributions
